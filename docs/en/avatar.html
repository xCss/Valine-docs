Avatar Setting

Edit On Github

Valine is used in Gravatar (with gravatar.cat.net mirror service) as commentator avatar.
Has seven optional values:

value style memo
Empty string:'' Gravatar official graphics
mm Mystic man (a grayhead)
identicon Abstract geometry(Generated by email or nickname)
monsterid little monster
wavatar A combination of different faces and backgrounds(Generated by email or nickname)
retro Eight-pixel retro portrait(Generated by email or nickname)
hide   Hidden avatar

The parameter value hide is supported at v1.1.8-beta.
If the value of avatar is not equal to one of the seven parameters, the default value ismm.

new Valine({
    ...
    avatar:'' // (''/mm/identicon/monsterid/wavatar/retro/hide)
});
